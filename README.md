# Neptun Plus!

## Egy új Neptun front-end androidhoz

- Felhasználóbarát
- Modern megjelenés
- Plusz funkciók (majd)

### Az app jelenleg pre-alpha stádiumban van, az alábbi funkciók működnek:

- Üzenetek
- Órarend
- Felvett tárgyak
- Felvett vizsgák
- Felvehető tárgyak megnézése
- Felvehető vizsgák megnézése
